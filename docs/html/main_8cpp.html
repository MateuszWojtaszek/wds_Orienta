int main ( int argc,

Inicjalizuje obiekt QApplication, który zarządza zasobami aplikacji Qt. Ustawia nazwę aplikacji. Tworzy instancję głównego okna aplikacji (MainWindow), wyświetla je na ekranie (w.show()), a następnie uruchamia główną pętlę zdarzeń Qt (QApplication::exec()), która obsługuje interakcje użytkownika i inne zdarzenia systemowe aż do zamknięcia aplikacji.

Parametry
argc[in] Liczba argumentów przekazanych do aplikacji z wiersza poleceń.
argv[in] Tablica ciągów znaków (C-style strings) zawierająca argumenty wiersza poleceń. Pierwszy argument (argv[0]) to zazwyczaj nazwa/ścieżka programu.
Zwraca
Kod wyjścia aplikacji przekazywany do systemu operacyjnego. Jest to wartość zwracana przez QApplication::exec(). Konwencjonalnie, 0 oznacza pomyślne zakończenie, wartości niezerowe wskazują na błędy.
