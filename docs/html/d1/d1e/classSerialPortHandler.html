Klasa obsługująca komunikację przez port szeregowy.

Odpowiada za otwieranie, zamykanie portu, odczytywanie danych, parsowanie ich do formatu QVector<float> oraz obsługę błędów.

◆ openPort()

bool SerialPortHandler::openPort ( const QString & portName,
qint32 baudRate = 115200 )

Otwiera i konfiguruje port szeregowy. Ustawia nazwę portu, prędkość transmisji (baud rate) oraz inne parametry komunikacji. Jeśli port był już otwarty, najpierw go zamyka. Czyści bufor wejściowy po pomyślnym otwarciu.

Parametry
portNameNazwa portu szeregowego do otwarcia (np. "COM3" lub "/dev/ttyUSB0").
baudRatePrędkość transmisji (domyślnie 115200).
Zwraca
Zwraca true jeśli port został pomyślnie otwarty, false w przeciwnym razie.
